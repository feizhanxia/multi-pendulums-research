# 数值模拟代码任务说明：全连接驱动摆的选择性振幅放大

## 总目标
- 搭建 $N=4$ 或 $5$ 个全连接、仅第 $1$ 个摆受周期驱动的阻尼单摆模型，验证在合适的驱动频率 $\Omega$ 与耦合强度 $K$ 下，非驱动某一摆的稳态振幅被显著放大而其他摆被抑制。
## 当前项目实现情况（已完成）
- 已实现 `simulate.py`：单次仿真 + summary 输出 + 可选 raw 保存 + θ(t) 图。
- 已实现 `scan.py`：Omega-K 网格扫描（支持多进程）+ summary 输出。
- 已实现 `analyze.py`：读取 summary 绘制热力图与 top-k 柱状图。
- 已实现 `model.py`：参数封装、ODE、初始条件与 RMS 指标。
- 已实现 `main.py`：集中配置并控制单次/扫描/分析流程。
- 已实现基础测试：`tests/test_pipeline.py` 覆盖单次仿真、扫描与绘图流程。

## 模型与方程
- 自由度：$\theta_i(t)$，$i=1..N$。
- 动力学：  
   $$
   \ddot{\theta}_i + \gamma \dot{\theta}_i + \omega_0^2 \sin\theta_i = \frac{K}{N}\sum_{j=1}^N \sin(\theta_j-\theta_i) + F\cos(\Omega t)\,\delta_{i1}
   $$
- 设 $\omega_0=1$（无量纲化）；仅 $i=1$ 受驱动。

## 代码实现要点
1) **积分器**：用 `solve_ivp`（RK45）或自写 RK4；设定足够长的积分时间，覆盖数百到上千个驱动周期以达稳态。  
2) **初始条件**：$\theta_i$、$\dot{\theta}_i$ 设为小随机扰动，避免对称解；固定随机种子以复现。  
3) **状态向量**：拼接 $[\theta_1..\theta_N, \omega_1..\omega_N]$，在 ODE 里拆分并计算耦合和驱动项。  
4) **参数接口**：集中配置 $\gamma, K, F, \Omega, N$，总时间 $T$，采样步长 $dt$，便于扫描；默认 $\omega_0=1$。  
5) **稳态截断**：跳过前若干驱动周期（例如前 30–50% 时间）后再统计振幅。  
6) **观测量**：每个摆稳态振幅 $A_i = \sqrt{\langle \theta_i^2 \rangle_t}$；可同时记录主频 FFT 幅值、相位。  
7) **选择性判据**：检查是否存在 $A_{\text{target}} \gg A_{\text{others}}$；保存满足阈值的 $(\Omega, K)$ 组合。  
8) **输出与可视化**：  
   - 时间序列与稳态窗口的 $\theta_i(t)$ 曲线。  
   - 振幅随 $(\Omega, K)$ 的热力图/曲线；标注最大放大摆与比值。  
   - 可选：相位差随时间、主频 FFT。

## 参数扫描建议
- 固定 $\gamma, F$（给出默认值，如 $\gamma\approx0.05$–$0.1$，$F\approx0.05$–$0.2$，可在代码里注释说明）。
- 扫描 $\Omega$：覆盖线性固有频率附近及耦合分裂带，例如 $0.5$–$1.5$，步长 $0.01$–$0.02$。  
- 扫描 $K$：从弱耦合到中等耦合，如 $0$–$1$（或更高），步长 $0.05$–$0.1$。
- 允许 $N=4$ 与 $N=5$ 的切换，观察模式结构变化。

## 下一阶段开发任务（新建代码，不修改现有实现）
### 研究目标（N=3 Proof-of-Concept）
- 驱动节点固定为 0；不再通过 $\omega_0$ 破缺对称；允许非对称、可正可负的 $K_{ij}$。
- 目标：在不同频段的 $\Omega$ 下，分别选择性激发节点 1 或节点 2，抑制其他节点。
- 频段由算法自动发现（不预设区间），偏向“强选择性比值”。

### 新增模块与文件（建议新建）
- `model_kij.py`：支持一般 $K_{ij}$（含非对称、可正可负）的 ODE 实现。
- `simulate_kij.py`：单次仿真接口，与现有 `simulate.py` 并行存在。
- `scan_kij.py`：扫描 Omega 与 K_ij 参数（或其参数化形式）。
- `analyze_kij.py`：\n+  - 频率扫描后的“节点胜出图”（Omega -> top-1 node）。\n+  - 自动频段发现与评估（区分度 + 选择性）。
- `search_kij.py`：粗搜索/随机搜索/局部微调的主入口（参数化 K_ij）。

### 任务清单（可按顺序推进）
1) 定义 $K_{ij}$ 参数化（N=3，6 个自由度，主对角为 0）。  
2) 实现 `model_kij.py` 并确认与现有模型的接口一致（Params/ODE/初始条件）。  
3) 实现 `simulate_kij.py`：输出 `amp_fft`、top-1 节点、选择性比值等指标。  
4) 实现 `scan_kij.py`：对 Omega 扫描；保存 per-Omega 指标。  
5) 实现 `analyze_kij.py`：

6) 实现 `search_kij.py`：  
   - 粗搜索（随机/网格） + 局部微调；
