# 多摆网络选择性激发：研究与开发规划（N=3）

## 研究目标
- 用全连接驱动摆网络（驱动节点固定为 0）实现“频段→节点”的选择性映射。
- 不依赖固有频率差异（omega_0 不破缺），而是通过非对称、可正可负的耦合矩阵 K_ij 调控能量传输路径与模态结构。
- 目标行为：
  - 在某一频段内，节点 1 的主频响应显著占优；
  - 在另一频段内，节点 2 的主频响应显著占优；
  - 不要求每个 Omega 都唯一对应一个节点，但在两个频段内要形成清晰分离。

## 开发目标
- **可解释的控制机制**：让驱动节点 0 在不同频率下更容易把能量送往 1 或 2。
- **自动频段发现**：算法自动找出最能区分节点 1/2 的两个频段，而非预设频段。
- **偏向强选择性**：优先追求“最大选择性比值”（top-1 / top-2 较大），稳健性不是首要目标。

## 初步开发整理（计划阶段）
### 1) 频率扫描与指标定义
- 扫描 Omega 范围（例如 0.6–1.4，步长 0.01–0.02）。
- 每个 Omega 计算稳态主频幅值 amp_fft（已有实现）。
- 记录：
  - top-1 节点索引（最大 amp_fft），
  - top-1 / top-2 比值（选择性）。

### 2) 自动频段发现（思路）
- 将频率轴切成多个候选区间（可滑动窗口或离散分段）。
- 目标函数示例：
  - 在频段 A 内，节点 1 为 top-1 的频率点比例高；
  - 在频段 B 内，节点 2 为 top-1 的频率点比例高；
  - 同时鼓励两个频段的平均选择性比值高。
- 从所有候选区间对中选出区分度最高的一对频段。

### 3) K_ij 参数化（N=3）
- 允许非对称和负值，先用 6 个自由度：
  - K_01, K_02, K_10, K_12, K_20, K_21（对角为 0）。
- 初期可使用结构化初始化（少数参数自由）以减少搜索空间，再逐步放开。

### 4) 搜索策略（初步）
- 先粗搜索：随机采样或粗网格（例如 [-1, 1] 范围）。
- 选出候选 K_ij 后局部细调，最大化“频段区分度 + 选择性”。

### 5) 验证与输出
- 输出：
  - 最优 K_ij 参数与对应频段；
  - 频率扫描的“节点胜出图”（Omega → top-1 节点）；
  - 选择性随 Omega 的曲线。
