# 回忆

有一对夫妻，大学时就在一起了。

那时他们都是穷学生，日子拮据，感情却热烈得发烫。男生身强体壮，肩背挺阔，走路带风，是那种年轻人特有的血气方刚；女生正值最鲜活的年纪，皮肤白净透亮，像晨光下带露的栀子，眼睛清澈有神，笑起来眉眼弯弯，脸颊会浮起淡淡红晕。她的美不是浓妆堆出来的，而是带着校园气息的灵动：干净、明亮、柔软，像一株刚抽芽的春枝，风一吹就有生命力。

因为没钱，他们常去学校外居民区小路边那种最便宜的小旅馆。门头褪色，招牌灯管时亮时暗；前台玻璃有细碎裂纹，登记本边角卷起；走廊窄而旧，墙皮起翘，灯泡发黄，空气里混着潮气、廉价香薰和消毒水的味道。房间里陈设简单：旧木门有点变形，床单洗得发薄，弹簧床一翻身就轻响，窗帘是褪了色的碎花布，墙角还留着岁月磨出来的斑驳印记。

他们总是开钟点房，为了省钱不过夜。每周两三次，他们急切又开心地从学校溜出来，熟门熟路钻进那间小旅馆。几个小时后再出来，手牵着手，面色红润，眼里还带着没散尽的笑意。有时他们还要匆匆赶回去上课；没课时，就并肩去吃路边摊，麻辣烫、小烤串、小火锅、猛火炒饭、烤生蚝，一边走一边聊，像把整条街都走成了青春的一部分。

后来他们毕业了，还是在一起。男生工作一段时间后开始创业，成了老板，事业渐有起色；女生进了大厂，赶上互联网浪潮，一路晋升，成了高P，俨然都市精英女性的模样。

他们的外貌也在岁月里悄悄变化。男生一直保持运动习惯，体态比从前更挺拔，脸部轮廓愈发清晰，少年时的稚气褪去，皮肤更紧致，整个人多了几分利落和锋芒。女生则从青涩走向成熟，少了学生气，多了成熟女性的明艳与从容；她依然漂亮，却不是浮夸的美，而是有分寸、有质感的风韵。大学时，他常是便宜T恤配运动鞋，她爱穿小碎花裙、长发披肩；如今，他的穿着依旧简洁，却更讲究版型与面料，低调里见品质；她的身材更显曲线，也不刻意张扬，却自然学会了高跟鞋与皮靴的分寸感。

他们的感情依然很好，是相伴多年后的熟悉与依恋。只是，也许因为太熟悉彼此，也许因为生活里那些说不清的复杂牵扯，两人对彼此的欲望不再像年轻时那样直接汹涌。并不是没有欲望，而是到了床上，总少了那种毫无保留、恨不得把彼此燃尽的冲动。

一次认真沟通后，他们想到了一个办法：偷偷开车去如今的大学城附近，再一起步行走进那些还没拆迁的小巷，去找当年的便宜旧旅馆。

当他们再次推开那样的房门，熟悉的窄走廊、发黄的灯光、陈旧的气味，像一把钥匙，突然打开了记忆深处最炽热的部分。那些被身份、业绩、会议和体面包裹住的自己，在那一刻松开了。他们重新找回年轻时的急切与贪恋，出来时又像当年那样亲密，连走路都靠得更近。

发现这一点后，他们有了只属于彼此的默契。每隔一段时间，只要一个眼神，两人就会暂时逃离名利场，逃离各自体面的社会身份，驱车去城市某个角落的便宜小旅馆。对他们来说，那些地方有时比最奢华的度假酒店更迷人。

这个秘密被他们安静地保存了很多年。后来身边的朋友、同事、下属，甚至来访的合作伙伴都发现：每隔一阵子，总有半天或几个小时，谁也联系不上他们，像是两个人一起消失了。至于原因，始终无人知晓。
